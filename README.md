# 🍔 Food Delivery Website (Frontend)

A modern food delivery frontend project built using **React.js** with a stylish and responsive UI.

---

## 🚀 Features

- ⚛️ Built using **React.js**
- 🎨 Stylish and modern user interface
- 🛒 Add and manage items in cart
- 🔐 User authentication (Login / Sign Up)
- 💳 Secure payment functionality
- 📦 Place and manage orders

---

## 🛠️ Tech Stack

- React.js
- Vite
- JavaScript (ES6+)
- CSS3

---

## 📦 Installation

```bash
git clone https://github.com/Niraj2one/Food-Delivery-Website.git
cd Food-Delivery-Website
npm install
